XML to Relation Mapping Browser


The XML to Relation Mapping Browser is a graphic tool with which users can view mapping information easily.

In the left panel a tree is used to display schema in a hierarchical format.  All elements of complex type is corresponding to a container node.  The elements of simple type and attributes are left nodes.  Users can quickly browse the hierarchical elements and access mapping information for an element or attribute by highlighting the corresponding tree node in the tree.

When a tree node is cliked on the left frame, its mapping informaiton is displayed in the right panel.  If the entity (element or attribute) is mapped to a table, the table is displayed.  If the entity is mapped to a field in a table, the field is highlighed in the corresponding table.
